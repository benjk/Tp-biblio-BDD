Le TP bibliothèque sur NodeJS permet une première utilisation de la base de données dans une appli Web.

Il reprend le dernier TP de Javascript simple mais cette fois, au lieu d'avoir les données brut, on va :
Afficher les données de la base
Insérer des nouveaux livres dans la base
Modifier des lignes existantes (colonne disponible)
